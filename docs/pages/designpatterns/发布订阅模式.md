# 发布订阅模式

## 介绍

发布订阅模式是一种一对多的关系，当一个对象的状态发生改变时，所有依赖于它的对象都将得到通知。

什么是发布订阅模式，其实小伙伴已经用到了发布订阅模式例如 addEventListener，Vue evnetBus

都属于发布订阅模式

## 应用场景

- 事件监听/广播
- Vue 的自定义事件
- ...

## 代码实现

```js
class Dispatch {
  // 考虑到多个事件，所以用对象存储
  #list = {};
  constructor() {
    this.list = {};
  }
  // 订阅
  on(name, callback) {
    // 如果没有该事件，就创建一个
    const callbackList = this.list[name] || [];
    callbackList.push(callback);
    this.list[name] = callbackList;
  }
  // 发布
  emit(name, ...args) {
    // 取出该事件的所有回调函数数组
    let evnetName = this.list[name];
    if (evnetName) {
      // 遍历执行
      evnetName.forEach((fn) => {
        fn.apply(this, args);
      });
    } else {
      console.error("该事件未监听");
    }
  }
  // 取消订阅
  off(name, fn) {
    // 取出该事件的所有回调函数数组
    let evnetName = this.list[name];
    if (evnetName && fn) {
      // 找到该回调函数的索引
      let index = evnetName.findIndex((fns) => fns === fn);
      // 删除该回调函数
      evnetName.splice(index, 1);
    } else {
      console.error("该事件未监听");
    }
  }
  // 一次性订阅
  once(name, fn) {
    // 用一个函数包裹
    let decor = (...args) => {
      fn.apply(this, args);
      // 执行后删除自身
      this.off(name, decor);
    };
    this.on(name, decor);
  }
}
```

## 测试

```js
let dispatch = new Dispatch();

dispatch.on("abc", (...arg) => {
  console.log(arg, 1);
});

dispatch.once("abc1", (...arg) => {
  console.log(arg, "once");
});

dispatch.emit("abc", true); // [true] 1

dispatch.emit("abc1", true); // [true] 'once'

dispatch.emit("abc1", true); // undefined
```
